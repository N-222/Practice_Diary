# Practice_Diary

# Дневник производственной практики

Простая программа для автоматизации учета выполненных работ во время прохождения производственной практики. Разработана в рамках модуля ПМ.02 «Осуществление интеграции программных модулей».

## Функционал
- Добавление записей о практике (дата, содержание работ, часы).
- Просмотр всех записей в удобном графическом интерфейсе.
- Удаление ошибочных или устаревших записей.
- Автоматический расчет общего количества отработанных часов.
- Сохранение данных в текстовый файл для предотвращения потери информации.

## Технологический стек
- Язык: Python 3.x
- Интерфейс: Tkinter (GUI)
- Хранение данных: diary.txt
